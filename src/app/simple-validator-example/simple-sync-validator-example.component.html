Simple Synchronous Form Control Validator

Three synchronous validator directives in this example

  • 1. Mark form control as required
  • 2. Require the length to be 3 characters or greater
  • 3. Check that the name entered is not equal to 'bob'
  • Value: {{ name.value }}
  • Status: {{ name.status }}
  • Errors: {{ name.errors | json }}
  • Pristine: {{ name.pristine }}
  • Touched: {{ name.touched }}
